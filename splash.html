The Handpicked jQuery Plugins Repository is a carefully selected roundup of some of the best jQuery Plugins ever released. Here we offer you the ability to browse through the plugins, or download a plugin that you fancy. Whether you're building a huge Web-Application, or a simple site, we've got you covered. We also offer you the ability to hot-link directly to these plugin scripts through our CDN service, so your site loads much faster.

We have disabled hot-linking directly to scripts in our online demos, however, below is a list of CDN URLs you should use if you're going to implement a plugin on your site. We are on a dedicated MaxCDN setup so all JS is Gzipped and served from 16 "edge locations" from around the world for peace of mind.

This Plugin library has been heavily modified and tweaked over the last few six months. Here's what I've done to it:

  • MADE the plugins 'runnable' / executable offline. Go ahead and download The Repo, and try to execute a script without an internet connection. All the scripts fallback to a local copy of jQuery when a remote version is not available.
  • ALL SCRIPTS now run on a jQuery 'baseline' version (v1.7.2), as jQuery is getting a major overhaul soon. We've decided to stop relying on grabbing the 'latest version' of jQuery for each script.
  • ALL PNGs are now optimized and compressed. Page load time is something we care about, a lot.
  • NEW influx of awesome scripts added. 2012 is a good year for jQuery, and Web Development in general. Long may it last.
  • ALL jQuery Script, and CSS files have been minified with YUI compressor. Again, page speed is something we care about. Feel free to 'beautify' a script if you feel it's too unreadable.
  • NEW site re-design. The page you are reading now has not always been here. Hooray for new iterations, and versions. I felt the old Splash Page was confusing people as to what the site is about.

You might want to read the blogpost concerning this repo, and the reasoning behind it. Remember, if you feel a plugin is missing from this, you can build your own repo using this site's code as a template. The aim of this repo is quality over quantity.

I hope you enjoy browsing through the list on the left, and remember, you can Download a plugin you like via our ZipFile directory. Do not try and scrape a plugin from our live server.
